# V.C.3: Assessment and evaluation methods

## Domain: Concepts in Instructional Design as Applied to Simulation (15%)

### [Status]: 🟢 Studied

### [Technician Perspective]
**Assessment and Evaluation Methods:** Use checklists, rubrics, or automated software scoring. Distinguish between Formative (feedback for learning) and Summative (grading for competency). Specialists ensure the technical reliability of these assessment tools.

### [Clinical Application]
Reliable assessment tools ensure that clinical competency is accurately measured before a learner works with real patients.

### [Key Terminology]
*   **Formative Evaluation:** 'Low stakes' feedback given during the learning process.
*   **Summative Evaluation:** 'High stakes' testing at the end of a unit or course.

### [Exam Scenario]
**Q:** You are running a summative exam and the automated scoring system fails. What is the operational backup?
**A:** Having a paper-based rubric or a secondary recording system ready so the facilitator can still assess the learner's performance.
